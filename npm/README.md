# lazyvec

A terminal UI for browsing vector databases — Qdrant, Pinecone, and Chroma (local + Cloud).

## Install

```bash
npm install -g lazyvec
```

Or run without installing:

```bash
npx lazyvec
```

On install, a prebuilt binary matching your platform is downloaded from the matching
[GitHub Release](https://github.com/armgabrielyan/lazyvec/releases) and verified against
`SHA256SUMS.txt`.

Supported platforms:

| OS | Architecture |
|----|--------------|
| macOS | x64, arm64 |
| Linux | x64, arm64 |
| Windows | x64 |

## Alternative install

```bash
curl -sSf https://raw.githubusercontent.com/armgabrielyan/lazyvec/main/install.sh | sh
```
